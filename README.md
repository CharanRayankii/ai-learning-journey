## 🚀 Quick Start

### 1. Clone this repository
```bash
git clone https://github.com/CharanRayankii/ai-learning-journey.git
cd ai-learning-journey
```

### 2. Set up your virtual environment
```bash
# Using conda (recommended for ML)
conda create -n ailearning python=3.9
conda activate ailearning

# OR using venv
python -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate  # Windows
```

### 3. Install dependencies (when added)
```bash
pip install -r requirements.txt
# OR
conda env create -f environment.yml
```

### 4. Start learning!
```bash
# Read the environment setup guide
cd fundamentals/environment-setup
# Follow venv-setup.md
```
