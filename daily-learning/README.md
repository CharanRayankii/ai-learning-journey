# 📁 Daily Learning Directory

Store your daily practice code and notes here, organized by date.

## Structure

```
daily-learning/
├── 2026-07-19/
│   ├── notes.md              # Key learnings from the day
│   ├── venv-practice.py      # Practice code
│   └── resources.md          # Links to resources used
│
├── 2026-07-20/
│   ├── data-types.py         # Daily practice
│   └── notes.md
│
└── 2026-07-21/
    └── ...
```

## How to Use

1. Create a new folder for each day: `YYYY-MM-DD`
2. Add your practice code and notes inside
3. Keep it organized by topic
4. Reference in DAILY_LOG.md

## Example Daily Structure

Each day should ideally have:
- **notes.md** - What you learned
- **practice.py** - Code examples
- **resources.md** - Links to resources used (optional)
