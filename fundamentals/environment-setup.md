# Environment Setup

## Step 1: Install Visual Studio Code
Download VS Code from the official page:

https://code.visualstudio.com/Download

Install it on your machine and open it once the installation is complete.

## Step 2: Install Anaconda
Download Anaconda from the official page:

https://www.anaconda.com/products/distribution

Install Anaconda on your machine and follow the installer instructions.

## Step 3: Install Python
Download Python from the official page:

https://www.python.org/downloads/

Install the version you need and make sure the installer adds Python to your PATH if prompted.

---

## Virtual Environment Setup
Before starting your project, set up a virtual environment for your Python dependencies.

Refer to the file `Venv_setup.md` and follow the steps there.
