## Quick Start

**If you want to start NOW (using conda - recommended):**
```bash
conda create -n ailearning python=3.9
conda activate ailearning
```

**If you prefer venv:**
```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
# or
venv\Scripts\activate  # Windows
```
